## 基本使用
1、单选框

```
<input type="radio" id="radio1" name="radio">
<label for="radio1" class="ui-radio"></label><label for="radio1">单选项1</label>
<!-- .ui-radio元素是看到的单选框效果，需要紧跟在原生的radio控件的后面 -->
```
<input type="radio" id="radio1" name="radio">
<label for="radio1" class="ui-radio"></label><label for="radio1">单选项1</label>

<input type="radio" id="radio2" name="radio">
